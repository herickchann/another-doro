# Release Notes — v1.0.12

## What's new

- **Settings drawer** — settings open in a slide-out panel instead of a modal, with a gear icon in the header
- **Session history sheet** — browse past sessions from a draggable bottom sheet; expand, collapse, or swipe to dismiss
- **Confirmation dialogs** — clear goals, clear history, and reset settings now ask before destructive actions
- **Goals polish** — break-time notice, clear-all control, and smoother goal management during sessions
- **Timer UI tweaks** — break label on the clock and refined layout across the main screen
